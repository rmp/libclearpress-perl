ClearPress Framework

Installing

ClearPress is available from CPAN. Installation is simplest via the CPAN shell: cpan ClearPress

Installation requires a handful of prerequisites, all available from CPAN. Notably some XML-related modules like XML::LibXML and database-related modules like DBD::SQLite and DBD::mysql.
